## 🚀 Project Structure

This documentation site is built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

```
.
├── public/             # Static assets (fonts, favicon)
├── src/
│   ├── assets/         # Images and logo
│   ├── components/     # Custom Astro components
│   ├── content/
│   │   └── docs/       # Documentation pages (Markdown/MDX)
│   │       ├── en/     # English documentation
│   │       └── zh/     # Chinese documentation
│   └── styles/         # Custom CSS styles
├── astro.config.mjs    # Astro configuration
└── package.json
```

## 📚 Documentation Structure

- **Start Here**: Introduction, getting started guide, core concepts
- **Core Features**: Traffic monitor, rules engine, AI assistant, developer tools
- **Advanced Usage**: Python scripting, plugin ecosystem
- **Reference**: Configuration, troubleshooting

## 🧞 Commands

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `pnpm install`            | Installs dependencies                            |
| `pnpm dev`                | Starts local dev server at `localhost:4321`      |
| `pnpm build`              | Build your production site to `./dist/`          |
| `pnpm preview`            | Preview your build locally, before deploying     |

## 🌍 Internationalization

This documentation supports multiple languages:

- **English** (`/`): Default language
- **简体中文** (`/zh/`): Chinese translation
