欢迎使用 RelayCraft！本指南将帮助您快速上手这款下一代 AI 原生网络流量调试工具。

## 安装

RelayCraft 支持 macOS、Windows 和 Linux 平台。

1.  访问我们的 [Releases 页面](https://github.com/relaycraft/relaycraft/releases) 或 [官方网站](https://relaycraft.dev/zh) 下载对应操作系统的最新版本。
2.  **macOS**: 打开 `.dmg` 文件并将 RelayCraft 拖入应用程序文件夹。
3.  **Windows**: 运行 `.exe` 安装程序并按照提示操作。
4.  **Linux**: 使用提供的 `.AppImage` 或 `.deb` 包。

## 初始化设置

首次启动 RelayCraft 时，它会在本地启动一个代理服务器（默认端口：`9090`）。

### 1. 配置代理

要检查来自您电脑的流量，您需要将网络流量路由到 RelayCraft。您可以选择配置 **系统代理** 或使用 **浏览器扩展**（推荐）。

-   **浏览器扩展 (推荐)**: 为了更精细的控制，建议使用如 **SwitchyOmega** 之类的浏览器扩展：
    -   新建情景模式 (例如 "RelayCraft")。
    -   代理协议: `HTTP`
    -   代理服务器: `127.0.0.1`
    -   端口: `9090` (或您配置的其他端口)。
    -   应用更改，并在需要抓包时切换到此情景模式。

-   **系统代理**: 当前版本暂未支持一键自动配置系统代理（计划在后续版本添加）。如果您需要抓取所有应用的流量，需手动在操作系统设置中配置代理服务器：
    -   代理协议: `HTTP`
    -   代理服务器: `127.0.0.1`
    -   端口: `9090`

### 2. 安装根证书

为了检查 **HTTPS** 流量（加密流量），您必须安装并信任 RelayCraft 根证书。

:::caution[重要]
如果未安装根证书，RelayCraft 将无法解密 HTTPS 流量，您将只能看到加密的 TCP 流或简单的 HTTP 请求。
:::

#### macOS / Windows

1.  点击侧边栏的 **证书** 图标（盾牌），或前往 **设置 > 证书管理**。
2.  点击 **安装并信任证书**。
3.  按照系统提示将证书添加到信任存储区。
    -   **macOS**: 您可能需要输入系统密码。如果未自动完成，请在“钥匙串访问”中确保该证书被设置为 **"始终信任"**。
    -   **Windows**: 安装程序应将其放入 "受信任的根证书颁发机构" 存储区。

#### iOS / Android (移动端抓包)

要检查来自移动设备的流量：

1.  **连接同一 Wi-Fi**: 确保您的电脑和移动设备处于同一网络下。
2.  **配置手机代理**:
    -   进入手机的 Wi-Fi 设置。
    -   选择当前连接的网络，找到 **代理** 设置。
    -   选择 **手动**。
    -   **主机名/服务器**: 输入电脑的局域网 IP 地址（在 RelayCraft 的 "代理配置指南" 中可见）。
    -   **端口**: `9090` (默认)。
3.  **下载证书**:
    -   在移动设备上打开浏览器（iOS 使用 Safari，Android 使用 Chrome）。
    -   访问 **[http://relay.guide](http://relay.guide)**。
    -   下载证书文件。
4.  **安装与信任**:

    **iOS**:
    1.  进入 **设置 > 已下载描述文件** 并安装描述文件。
    2.  进入 **设置 > 通用 > 关于本机 > 证书信任设置**。
    3.  开启 **RelayCraft CA** 的开关（启用完全信任）。

    **Android**:
    1.  进入 **设置 > 安全/加密与凭据 > 安装证书 > CA 证书**。
    2.  选择刚才下载的文件并确认安装。
    3.  *注意：Android 11+ 系统可能有更严格的限制。某些启用了 SSL Pinning 的应用可能无法抓包。*
