您可以通过点击侧边栏或屏幕底部的 **设置 (齿轮)** 图标来访问设置面板。

## 网络

### 代理设置
-   **端口**: RelayCraft 监听的本地端口 (默认: `9090`)。
-   **SSL 代理**: 启用/禁用 HTTPS 拦截。
-   **上游代理**: 如果您处于公司代理之后，请在此配置。
    -   **URL**: 例如 `http://corp-proxy:8080`
    -   **绕过域名**: *(当前版本暂不支持)* 逗号分隔的域名列表，用于绕过上游代理 (例如 `localhost, 127.0.0.1`)。

### 证书
-   **根证书状态**: 显示 RelayCraft CA 是否受操作系统信任。
-   **安装/重新安装**: 帮助您在系统上安装证书或为移动设备导出证书的按钮。
-   **重新生成**: 创建一个新的根 CA (注意: 这将使所有以前信任的证书失效)。

## 外观

-   **主题**: 浅色、深色或系统默认。
-   **显示密度**:
    -   *紧凑*: 更多行，更少填充。
    -   *舒适*: 平衡 (默认)。
    -   *宽松*: 更多间距。
-   **语言**: 英语、简体中文。

## 常规

-   **开机启动**: 计算机启动时自动运行 RelayCraft。
-   **关闭行为**: 关闭窗口时最小化到托盘或退出应用程序。
-   **更新**: 手动检查更新。点击检查后若有新版本会提示升级（不支持自动检查）。
-   **始终置顶**: 保持 RelayCraft 窗口位于其他应用程序之上。

## 插件

-   **市场**: 浏览和安装插件。
-   **已安装**: 管理您的活动插件。
-   **从文件安装**: 支持从本地文件直接安装插件包。
