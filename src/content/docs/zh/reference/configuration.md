title: 配置

---

您可以通过点击侧边栏或屏幕底部的 **设置 (齿轮)** 图标来访问设置面板。

## 常规

-   **语言**: 切换界面语言（English / 简体中文）。
-   **始终置顶**: 保持 RelayCraft 窗口位于其他应用之上。
-   **退出前确认**: 关闭应用时弹出确认框，避免误操作。
-   **自动开始捕获**: 启动应用时自动开始抓包。

## 网络与代理

-   **代理端口**: 本地代理监听端口（默认 `9090`，范围 `1024-65535`）。
-   **忽略 SSL 证书校验**: 适用于内网/自签名证书等调试场景。
-   **上游代理**: 配置 RelayCraft 对外联网时使用的代理。
    -   **代理 URL**: 例如 `http://127.0.0.1:7890`
    -   **连接检测**: 可直接测试上游代理可达性。

## MCP 服务器

-   **启用 MCP 服务器**: 启动/停止内置 MCP 服务端点。
-   **MCP 端口**: 默认 `7090`。
-   **Bearer Token**: 用于 MCP 写操作工具（`create_rule`、`replay_request`、`delete_rule`、`toggle_rule`）。
-   **配置片段**: 一键复制客户端接入配置（Claude Desktop、Cursor 等）。

## 证书

-   **信任状态**: 显示操作系统是否已信任 RelayCraft 根 CA。
-   **安装并信任 / 移除**: 安装或移除系统信任链中的根证书。
-   **重新生成根 CA**: 重新生成证书与私钥。
-   **打开证书目录**: 快速定位本地证书文件。

## 外观

-   **主题**: 浅色、深色或系统默认。
-   **显示密度**:
    -   *紧凑*: 更多行，更少填充。
    -   *舒适*: 平衡 (默认)。
    -   *宽松*: 更多间距。

## 插件

-   **市场**: 浏览和安装插件。
-   **已安装**: 管理您的活动插件。
-   **从文件安装**: 支持从本地文件直接安装插件包。

## 高级与诊断 / 关于与支持

-   **查看系统日志**: 打开内置日志查看器（代理/系统/脚本/插件/崩溃日志）。
-   **详细模式（Verbose）**: 输出更详细运行日志，便于深度排障（修改后需重启生效）。
-   **打开配置/数据/日志目录**: 直接跳转 RelayCraft 运行目录。
-   **检查更新**: 手动检查最新版本并执行升级。
