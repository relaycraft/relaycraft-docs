RelayCraft 拥有强大的 **插件生态系统**，允许您扩展核心功能之外的用户界面和功能。Python 脚本负责流量拦截逻辑，而 UI 插件则允许您构建自定义仪表板、数据可视化工具或集成工具。

## 插件市场

最简单的入门方式是通过 **插件市场**。
1.  前往 **设置** > **插件** > **市场**。
2.  浏览来自社区的可用插件。
3.  点击 **安装** 将插件添加到您的工作区。

## 插件能力

RelayCraft 中的插件本质上是在应用程序上下文中运行的 **React 组件**。它们可以：
-   **可视化数据**: 根据流量数据创建自定义图表或图形。
-   **解码格式**: 为专有数据格式（例如 Protobuf, gRPC）添加查看器。
-   **集成工具**: 将 RelayCraft 连接到您团队的问题跟踪器或文档系统。

## 管理插件

在 **设置** > **插件** 面板中，您可以：
-   **启用/禁用**: 开启或关闭插件，无需卸载。
-   **配置**: 某些插件会公开自己的设置（例如 API 密钥、显示首选项）。
-   **卸载**: 删除不再需要的插件。
